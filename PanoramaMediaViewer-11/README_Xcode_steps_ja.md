# PanoramaMediaViewer

iPad用の簡易360度写真・360度動画ビューアです。

## MVP仕様

- ファイルアプリ上の任意フォルダから読み込み
- 写真と動画を同じ一覧に表示
- サムネイルなし
- 拡張子は非表示
- 表示形式
  - `写真 THETA_0001`
  - `動画 THETA_0002`
- 撮影日時順ソートなし
- 検索欄なし
- 横幅に応じて複数列表示
- Split View対応は必須要件から外し、原則フルスクリーン利用を想定
- タップで360度ビューアを開く
- 指ドラッグで視点回転
- ピンチでズーム
- 動画は自動再生し、下部ボタンで再生・一時停止
- 動画シークバーで任意の時間へ移動
- 動画シークバーをドラッグ中も映像更新を試みる

## 対応形式

- 写真: jpg, jpeg, png, heic, heif, tif, tiff
- 動画: mp4, mov, m4v

入力元はファイルアプリ上の任意フォルダのみです。
写真アプリからの読み込み機能は廃止しています。

## Xcodeで開く方法

1. ZIPを展開する
2. `PanoramaMediaViewer.xcodeproj` をXcodeで開く
3. 左上の実行先をiPadシミュレータ、または接続したiPadにする
4. Signing & CapabilitiesでTeamを自分のApple IDまたは開発チームに設定する
5. Bundle Identifierが他人と重複する場合は変更する
6. Runを押す

## 使い方

1. アプリを起動する
2. `フォルダを選択` を押す
3. ファイルアプリ上で360度写真・360度動画を保存したフォルダを選ぶ
4. 一覧から `写真` または `動画` の項目をタップする
5. 360ビューアで表示する

## Split Viewについて

Split View対応は必須要件から外しました。
`Info.plist` では `UIRequiresFullScreen` を `true` にしています。
XcodeのTarget設定で `Requires full screen` が無効になっている場合は有効にしてください。

ただし、iPadOSのバージョンやビルド環境によっては、システム側の仕様により完全なSplit View除外を保証できない場合があります。

## 注意

- このMVPは正距円筒投影、つまり横長の360画像・動画を前提にしています。
- RICOH THETA等で未変換の二眼魚眼形式になっているファイルは、ストリートビュー風に正しく表示されない可能性があります。
- フォルダ読込は選択したフォルダ直下のみを対象にしています。サブフォルダ再帰検索は未実装です。
- App Store配布用ではなく、まず実機確認するためのMVPです。

## 修正版の確認項目

動画を開いた後、以下を確認してください。

1. 一覧画面に写真アプリボタンが表示されないこと。
2. `フォルダを選択` のみで読み込むこと。
3. 画面をドラッグしたとき、以前と反対方向に視点が動くこと。
4. 動画再生中に指を離していても、映像が進み続けること。
5. 画面下部のシークバーで任意の時刻へ移動できること。
6. シークバーをドラッグしている間も、可能な範囲で映像が追従すること。
7. 再生 / 一時停止ボタンが機能すること。


## v5 変更

- 一覧表示をファイル名順にしました。
- `THETA_0001`, `THETA_0002` のような連番ファイルが上から順に並ぶ想定です。


## v6 変更

- Split View対応を必須要件から外し、`UIRequiresFullScreen` を `true` に変更しました。
- フォルダ選択時のSecurity Scoped Resourceを、再選択時・解放時に停止するよう整理しました。

## v7の変更点

- iPhone / iPad両対応に変更しました。
- Xcodeの `TARGETED_DEVICE_FAMILY` を `1,2` に変更しました。
- iPhone向けの画面向き設定を追加しました。
- iPhone幅でも一覧が崩れにくいよう、一覧項目の最小幅を 190pt から 160pt に変更しました。
- Split View非対応方針は維持し、`UIRequiresFullScreen` は `true` のままです。

## v8の変更点

- 動画再生時の下部コントロールを2行から1行に変更しました。
- 再生 / 一時停止、シークバー、時間表示を同じ行にまとめました。
- 映像面積を広げるため、コントロールを小さめの半透明オーバーレイにしました。
